Sobre mim

Também estou desenvolvendo aplicações com HTML, CSS e JavaScript. No primeiro semestre aprendi a programar com Python e Portugol. Nesse tempo fiz cursos na Alura focados em HTML , CSS e Orientação à Objeto com Python.

O projeto visava atender as necessidades das equipes de manuteção de uma empresa fictícia. A necessidade era em relação a reservas de ferramentas específicas para utilização dos técnicos. Este projeto está disponível no meu perfil do Github ,assim como diversas aplicações em Python, HTML e CSS que fiz para poder aprofundar meus conhecimentos nessas linguagens.

Pretendo migrar minha área de atuação de mecânico de usinagem para ser desenvolvedor, estou a procura de uma oportunidade.

Além das certificações e conhecimentos em programação eu sou muito proativo, com uma boa comunicação oral e escrita, sou muito empático com as pessoas, gosto de estar estudando constantemente e também gosto de enfrentar desafios, pois eu acredito que a cada desafio enfrentado eu tenho um ganho enorme em aprimoramento pessoal.
